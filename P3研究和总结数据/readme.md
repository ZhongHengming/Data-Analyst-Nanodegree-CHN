**项目背景：** 在此项目中，将使用 R 并运用探索性数据分析来探索一个变量或多个变量之间的关系，以及在一个选定的数据集中探索分布、异常值和反常现象。

**项目数据源：** 

- `wineQualityReds.csv`：红葡萄玖质量

这个整⻬的数据集包含1,599 种红酒，以及 11 个关于酒的化学成分的变量。⾄少 3 名葡萄酒专家对每种酒的质量进⾏了评分，分数在 0（⾮常差）和10（⾮常好）之间。
